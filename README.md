# sax
Symbolic Aggregate Approximation
Dimension reduction - Knitr and Shiny app


### Shiny app

in order to run the shiny app, start an R session and type the following commands:
```
require(shiny)
runApp()
```

#### Dependencies
```
require(shiny)
require(ggplot2)
require(knitr)
require(seewave)
require(Quandl)
require(rCharts)
require(TSclust)
require(stats)
require(dplyr)
require(wavelets)
```
